# Scenario-Planning-Financial-Model-Base-Best-Worst-Scenarios

A dynamic Excel financial model for scenario planning and forecasting, allowing users to select Base, Best, or Worst cases and see real-time impacts on Income Statement, Balance Sheet, and Cash Flow.

Purpose

Evaluate financial performance under multiple scenarios

Support decision-making, budgeting, and risk management

Forecast cash flow, working capital, and key financial ratios

Key Features

Scenario Toggle: Switch between Base, Best, and Worst scenarios to instantly update assumptions, operational drivers, and all financial statements.

Integrated 3-Statement Model: Fully linked Income Statement, Balance Sheet, and Cash Flow for consistency and accuracy.

Operational Drivers: Converts units, pricing, mix, and inventory assumptions into financial results.

Headcount & OPEX Projection: Projects staffing, salaries, benefits, and operating expenses.

Dynamic Summary: Consolidated outputs with growth metrics, performance visualization, and leverage ratios.

Decision Support: Provides actionable insights for resource allocation, strategic planning, and scenario analysis.
